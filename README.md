# VittaWear
O VittaWear é um projeto de e-commerce desenvolvido com foco em performance, responsividade e experiência do usuário. A aplicação conta com uma landing page moderna e adaptável, construída utilizando HTML, CSS e JavaScript, garantindo compatibilidade com diferentes dispositivos.  No backend, o sistema é estruturado em Python, oferecendo robustez e escalabilidade para o gerenciamento das funcionalidades da plataforma. O armazenamento de dados é realizado por meio de um banco de dados MySQL, assegurando organização, integridade e eficiência no tratamento das informações.  O projeto tem como objetivo integrar uma interface intuitiva com uma arquitetura sólida, proporcionando uma base completa para operações de comércio eletrônico.
